# HyperXray Product Overview

HyperXray is a high-performance proxy client for Android built on Xray-core (XTLS/Xray-core).

## Core Value Proposition
- Embeds Xray-core directly in `libhyperxray.so` native Go library
- Provides WireGuard over Xray tunneling (WARP integration)
- AI-powered connection optimization using ONNX models
- Modern Material 3 UI with Jetpack Compose

## Key Features
- VPN service with multiple protocol support (VLESS, Reality, Hysteria2, WireGuard/WARP)
- Intelligent routing with geoip/geosite rules
- On-device ML for connection optimization (LinUCB bandit, policy models)
- Telegram bot integration for remote monitoring
- Per-app VPN routing

## Target Platform
- Android 10+ (API 29+)
- Architectures: arm64-v8a, armeabi-v7a, x86_64

## Package
- Application ID: `com.hyperxray.an`
- Distribution: GitHub Releases, F-Droid
